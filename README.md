# Student Expense Tracker (Java)

This project was developed as part of my BYOP assignment. 
Initially, I faced issues while handling multiple Java files and debugging errors, 
but I learned how to structure a project properly and use GitHub for version control.


## 📌 Problem

Students often fail to track daily expenses, leading to poor financial management.

## 💡 Solution

This project is a Java-based console application that helps users track and manage daily expenses.

## 🚀 Features

* Add Expense
* View Expenses
* View Total Expense
* Simple command-line interface

## 🛠 Technologies Used

* Java
* Object-Oriented Programming
* ArrayList

## ▶️ How to Run

1. Compile:
   javac *.java

2. Run:
   java Main

## 📚 Learning Outcomes

* Learned Java OOP concepts
* Built a real-world application
* Learned GitHub usage and version control
